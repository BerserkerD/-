## 西电微机原理课程设计
课题五：点阵式 LCD 动态显示系统设计

通过键盘操控，在液晶屏上实现西安地铁六号线的站名切换。具体实现下一站、到站的切换。上下行切换、插入广告，滚动播报下一站站名。

键盘0：上下行切换 ，键盘1：进一站，键盘2：到站

键盘4：插播广告，键盘5：退一站，键盘6：出站
